# Rotary Encoder - Interface with STM32 (NUCLEO-F103RB)





### Wiring Diagram
```
 Encoder Signal  Wire Color  NUCLEO Pin  STM32 Signal

	DT            Brown        CN7-30      PA1
	SW            Yellow       CN7-38      PC0
	+             Red          CN7-16      3V3
    GND           Black        CN7-20      GND
```
